# HanziKey Website

Support and privacy pages for [HanziKey](https://apps.apple.com/), a
pop-up Mandarin dictionary for macOS.

## Pages

- [Support](https://yourusername.github.io/hanzikey-website/) — `index.md`
- [Privacy Policy](https://yourusername.github.io/hanzikey-website/privacy) — `privacy.md`

## Deployment

Hosted on GitHub Pages. Pushes to `main` deploy automatically.

To preview locally:

```bash
bundle install
bundle exec jekyll serve
```
